# 🧠 LearnAI — AI-Powered Learning Assistant

> A full-stack learning app built with the MERN stack + Groq AI (Llama 3) — 100% Free!

---

## 📌 About The Project

**LearnAI** is a full-stack AI-powered learning assistant that helps users learn coding and languages through:

- 💬 **AI Chat Tutor** — Ask questions and get real AI answers instantly
- 🧪 **Quiz Generator** — Enter any topic and get a custom quiz generated by AI
- 🗂️ **Flashcard Generator** — Generate flashcards on any topic with one click
- 📊 **Dashboard** — Track your learning progress and activity

---

## 🎯 Features

| Feature | Description |
|---|---|
| 💬 AI Tutor | Real-time AI tutoring powered by Groq + Llama 3 |
| 🧪 Quiz Generator | Auto-generates 5-question quizzes on any topic |
| 🗂️ Flashcard Generator | Creates 8 flashcards per topic with flip animation |
| 📊 Dashboard | Tracks questions asked, quizzes taken, flashcards made |
| 🔐 Authentication | JWT login/register with bcrypt password hashing |
| 💾 Data Persistence | All data saved to MongoDB Atlas cloud database |

## ⚙️ Getting Started

### Prerequisites
- Node.js v18+
- Git

### Step 1 — Get Free API Keys

**Groq API Key (Free)**
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up → API Keys → Create Key
3. Copy the key (starts with `gsk_`)

**MongoDB URI (Free)**
1. Go to [mongodb.com](https://mongodb.com)
2. Sign up → Create free M0 cluster
3. Connect → Drivers → copy the URI

---

### Step 2 — Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Learning-assistant
cd learnai-groq
```

---

### Step 3 — Backend Setup

```bash
cd backend
cp .env.example .env
```

Open `.env` and fill in your keys:
```env
MONGO_URI=your_mongodb_uri_here
GROQ_API_KEY=your_groq_api_key_here
JWT_SECRET=any_random_string_here
PORT=5000
```

```bash
npm install
npm run dev
```

Backend runs on → `http://localhost:5000`

---

### Step 4 — Frontend Setup

Open a new terminal:

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on → `http://localhost:5173`

## 🔌 API Endpoints

| Method | Endpoint | Description | Auth |
|---|---|---|---|
| POST | `/api/auth/register` | Create account | No |
| POST | `/api/auth/login` | Login, get JWT | No |
| POST | `/api/chat/ask` | Ask AI a question | Yes |
| GET | `/api/chat` | Get chat history | Yes |
| POST | `/api/quiz/generate` | Generate quiz with AI | Yes |
| POST | `/api/quiz/submit` | Submit answers, get score | Yes |
| GET | `/api/quiz` | Get quiz history | Yes |
| POST | `/api/flashcard/generate` | Generate flashcards | Yes |
| GET | `/api/flashcard` | Get all flashcards | Yes |
| PATCH | `/api/flashcard/:id` | Toggle mastered | Yes |

## 🚀 How It Works

```
User types question
       ↓
React frontend (Vite)
       ↓
Axios POST → Express API (Node.js)
       ↓
Groq AI API (Llama 3 — Free)
       ↓
AI Response saved to MongoDB
       ↓
Response sent back to React
       ↓
Answer displayed to user
```
